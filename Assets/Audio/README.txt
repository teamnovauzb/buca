CLEANUP
═══════════════════════════════════════════════════════════

The Assets/Audio/_downloads/ folder contains the original Kenney ZIPs
+ extracted contents (~6 MB). It's safe to delete — every file we
actually use is already copied into the category folders above.

If "Device or resource busy" prevents deletion (OneDrive lock),
quit Unity, wait for OneDrive to finish syncing, then delete
manually. None of the project's audio references point inside it.
